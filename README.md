# Doctor Finder Template

## Overview
**Doctor Finder Template** is a **frontend-only web template** designed to help users **find and book doctors** easily. Built with **HTML, CSS,Tailwindcss, and JavaScript**, it provides a clean, modern, and responsive interface.  

---

## Features
- **Doctor Listing**   
- **Appointment Booking Form** 
- **Responsive Design**
